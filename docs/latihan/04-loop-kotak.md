# Latihan Program

Buat program menanyakan _panjang_ dan _lebar_. Setelah itu, program membuat "kotak" yang terdiri dari karakter "\*" dengan _panjang_ sebagai jumlah baris dan _lebar_ sebagai jumlah kolom.

**Contoh 1**
```
panjang? 3
lebar? 4

****
****
****
```

**Contoh 2**
```
panjang? 4
lebar? 3

***
***
***
***
```
